
De site: 
1. een digitaal visitekaartje naar alle kanten toe met relevante en representatieve content over Bee LKQ,
2. een template project voor deelnemers.

Gebouwd met alleen css en html. Voor webtesters zeer relevante talen en zeker ook voor web frontend mensen.
Wie deze template wil gebruiken leert eerst het handwerk. Html en css zijn prima instappers. 
Je leert twee redelijk eenvoudige computertalen die je helpen om andere talen sneller te leren.     

Mappenstructuur en indexpagina:
In de main van de indexpagina zijn de links per onderwerp gegroepeerd. Die indeling zie je ook 
in de map html terug. 

De media in de mappenstructuur zijn op soort ingedeeld.
